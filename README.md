# ⭐ StarMap Workshop — Example Scripts

Welcome to the **StarMap Workshop repository**!  
This repo contains **hands-on examples and starter scripts** used during the workshop to help you explore, process, and visualize large geospatial datasets.

---

## 📍 About the Workshop

The **StarMap Workshop (UC Riverside)** introduces participants to:

- 🌍 Working with large-scale geospatial datasets
- 🧩 Vector tiles (MVT) and tiled map systems
- ⚡ Efficient data processing pipelines
- 🎨 Interactive map visualization and styling
- 🤖 LLM-assisted map styling (experimental)

This repository is designed to be **practical and beginner-friendly**, with minimal setup and clear examples.
